Enhancement Overview

Capstone Project Overview

For my CS499 Capstone project, I selected the Weight Tracker application that I originally developed in CS360: Mobile Architecture and Programming. The original application provided a working foundation for user authentication, weight tracking, goal management, and notifications, but there were several opportunities to improve the overall design, organization, scalability, and user experience.

Rather than selecting multiple artifacts, I chose to use a single project throughout all three enhancement categories. This approach allowed me to demonstrate how an application can evolve over time through incremental improvements while maintaining its original purpose. Each enhancement built upon the work completed in the previous category, resulting in a more complete and realistic Android application.

  • CS360 Weight Tracker App

Category One: Software Design and Engineering

The first enhancement focused on improving the overall architecture, maintainability, and user experience of the application. Major improvements included a redesigned authentication system, dashboard redesign, account management features, settings functionality, unit conversion preferences, and the implementation of a more scalable DailyRecord architecture.

These changes transformed the application from a basic course project into a more organized and maintainable software solution while improving the overall user experience.

  • Category One: Software Design and Engineering Enhancement

Category Two: Algorithms and Data Structures

The second enhancement focused on improving how historical data is organized, retrieved, and presented to users. A new History screen was developed that includes dynamic calendar generation, monthly navigation, record highlighting, historical record retrieval, and analytics calculations.

This enhancement demonstrated searching, filtering, aggregation, and data visualization concepts while providing users with meaningful insight into their historical progress.

  • Category Two: Algorithms and Data Structures Enhancement

Category Three: Databases

The final enhancement focused on expanding the application's SQLite database functionality. A new motivational_messages table was added to support database-driven motivational content. The enhancement included schema expansion, database version upgrades, data initialization workflows, and randomized query retrieval.

These additions expanded the role of the database beyond simple data storage and demonstrated practical database integration within a mobile application.

  • Category Three: Database Enhancement

Final Result

By the conclusion of the Capstone project, the Weight Tracker application had evolved significantly from the original CS360 artifact. The final application features improved software architecture, enhanced user experience, historical data visualization, expanded database functionality, stronger account management capabilities, and a more scalable overall design.

Together, these enhancements demonstrate growth across software engineering, algorithms and data structures, and database design while showcasing the skills developed throughout the Computer Science program.
