Welcome to My Computer Science Capstone ePortfolio

This portfolio showcases the work I completed throughout the Computer Science program at Southern New Hampshire University, culminating in my CS499 Capstone project. The centerpiece of this portfolio is a Weight Tracker mobile application that I originally developed in CS360: Mobile Architecture and Programming and later enhanced as part of the Capstone experience.

Through a series of software engineering, algorithm, and database enhancements, I transformed the original application into a more scalable, feature-rich, and user-friendly solution. Along the way, I gained valuable experience working with software architecture, data management, database integration, testing, and technical documentation.

This portfolio highlights that journey from the original application to the completed Capstone project. It includes the original artifact, code review, enhancement narratives, technical documentation, and the final version of the application.

Thank you for taking the time to visit my portfolio. I invite you to explore the project, review the enhancements, and see how my skills developed throughout the Computer Science program.
